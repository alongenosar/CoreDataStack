  • CoreDataStack

    CoreDataStack is a safe, simple, and easy-to-use Core Data stack. The focus is on making safe Core Data the easy path, thereby avoiding common bugs associated with using multiple contexts.

    CoreDataStack performs modifications in a background queue, and offers read-only access to the main thread.
